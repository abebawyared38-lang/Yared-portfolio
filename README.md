# E-Commerce Hub (Full-Stack Web Application)

A full-stack e-commerce web application built using React, Node.js, and MySQL.  
The project demonstrates how a frontend interacts with backend APIs and a relational database to deliver dynamic product data.

---

## 🚀 Features

- Displays product listings dynamically
- View detailed product information
- Backend API integration for data handling
- Structured database for product storage
- Responsive user interface

---

## 🧠 System Architecture

### Frontend (React)
- Built reusable components for product listing and display
- Managed application state for dynamic UI updates
- Handled user interactions such as viewing product details

### Backend (Node.js + Express)
- Created RESTful API endpoints for product data
- Managed communication between frontend and database
- Handled server-side logic for data retrieval

### Database (MySQL)
- Stored structured product data in relational tables
- Enabled efficient querying of product information

---

## 🔄 System Flow

User → React Frontend → Node.js API → MySQL Database → Response → UI Update

---

## 🛠️ Tech Stack

- React
- Node.js
- Express.js
- MySQL
